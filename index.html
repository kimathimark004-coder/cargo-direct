Installation

Install Cargo Direct from crates.io with one command.

cargo install cargo-direct

Usage

Add a dependency directly from the terminal.

cargo direct add serde

Cargo Direct is designed to be predictable: common actions have short, readable commands.

About

Cargo Direct is an open-source project built using Rust to simplify dependency management for Cargo-based projects. It’s meant to feel like a natural extension of your existing workflow—fast, focused, and terminal-first.
